# 🚀 WebWatch - Study Clock & Focus Timer

A modern, feature-rich web application designed to help you track focus time, manage study sessions with Pomodoro technique, and monitor your productivity with beautiful animations and intuitive controls.

## ✨ Features

### ⏱️ **Timer Modes**
- **Focus Mode** (25 minutes) - Deep work sessions with goal tracking
- **Short Break** (5 minutes) - Quick recovery time
- **Long Break** (15 minutes) - Extended rest after multiple sessions

### 🌍 **Clock Display**
- Digital clock with seconds
- Beautiful SVG analog clock with smooth hand movements
- Real-time date and personalized greeting
- Dark/Light theme support

### 📊 **Statistics & Analytics**
- Track total focus time
- Count completed sessions
- Monitor daily goal progress (default: 2 hours)
- View longest session record
- Weekly chart showing session distribution by day
- Auto-reset statistics at midnight

### ⚙️ **Customization**
- Adjustable timer durations
- Custom daily focus goals
- Sound notifications (toggle on/off)
- Auto-start breaks after focus sessions
- Focus Mode for distraction-free studying
- Theme switcher (Dark/Light)

### 🎨 **User Experience**
- Smooth animations and transitions
- Progress ring with real-time updates
- Toast notifications for actions
- Keyboard shortcuts for quick control
- Responsive design for mobile & desktop
- Data persistence via LocalStorage
- Auto-save every 30 seconds

## 🎮 **How to Use**

1. **Open the App**: Simply open `index.html` in your web browser
2. **Select Timer Mode**: Click on Focus, Break, or Long Break
3. **Start Session**: Click the Start button or press Space
4. **Monitor Progress**: Watch the circular progress ring and timer
5. **Complete Session**: Timer auto-completes, can auto-start break
6. **View Stats**: Click the 📊 button to see statistics panel
7. **Adjust Settings**: Click ⚙️ to customize durations and preferences

## ⌨️ **Keyboard Shortcuts**

| Key | Action |
|-----|--------|
| **Space** | Play/Pause Timer |
| **R** | Reset Timer |
| **S** | Skip Session |

## 🎯 **Pomodoro Technique**

The classic Pomodoro method is built-in:
1. Work for 25 minutes (Focus)
2. Take a 5-minute break (Short Break)
3. Repeat 3-4 times
4. Take a 15-minute break (Long Break)

All timings are customizable in Settings!

## 🐛 **Troubleshooting**

### Timer not starting?
- Check if timer is already running
- Try clicking Reset first
- Make sure you enabled JavaScript

### Sound not working?
- Check if browser has audio permissions
- Enable sound in Settings
- Some browsers require user interaction first

### Stats not saving?
- Check if LocalStorage is enabled
- Clear browser cache and reload
- Try a different browser

### Clock not updating?
- Refresh the page
- Check system time is correct
- Try restarting the browser
